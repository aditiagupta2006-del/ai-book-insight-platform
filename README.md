# 📚 AI-Powered Book Insight Platform

## 🚀 Project Overview

This project is an AI-powered web application that processes book data and provides intelligent insights using Natural Language Processing (NLP) and a RAG (Retrieval-Augmented Generation) system.

---

## ✨ Features

* 📚 Book Upload (User Input)
* 📊 Dashboard (Book Listing)
* 📖 Book Details Page
* 🤖 AI Features:

  * Summary Generation
  * Genre Classification
  * Sentiment Analysis
* ❓ Question Answering (RAG)
* 🔍 Book Recommendation System

---

## 🧠 AI Implementation

* Sentence Transformers for embeddings
* ChromaDB for vector storage
* RAG Pipeline:

  * Query → Embedding → Search → Answer

---

## 🛠️ Tech Stack

* Python
* Transformers
* Sentence Transformers
* ChromaDB
* Gradio
